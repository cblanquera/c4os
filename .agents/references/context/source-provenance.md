# Context Source Provenance

Status: active
Created: 2026-06-21
Updated: 2026-06-21

This reference keeps source paths and artifact provenance out of
`.agents/context/` entries while preserving enough routing detail for future
agents.

## Imported Human Planning Sources

- `plans/product-brief.md`: product direction, target users, goals, product
  model, feature surfaces, constraints, and restart intent.
- `plans/product-interface.md`: current interface contract, three-panel shell,
  composer behavior, settings navigation, right-panel behavior, and UI
  interaction rules.
- `plans/pegs/*.png`: visual peg sources for UI direction.

## Root Artifacts

- `wireframes/screens.md`: wireframe routing and screen notes.
- `wireframes/ui-handoff-spec.md`: accepted r04 UI handoff details used by the
  MVP spec and traceability records.
- `wireframes/r04-single-page-app/`: reviewable r04 functional artifact.
- `proofs/`: POC implementation artifacts and proof-local evidence.

## Agent Records

- `.agents/specs/research/`: imported research records, validation, POC
  results, and research closeout.
- `.agents/specs/mvp/`: current MVP contract draft derived from frozen
  research records.
- `.agents/context/product-specs.md`: context-level feature inventory
  promoted from the current MVP contract and accepted r04 handoff.
- `.agents/references/research/grill-session.md`: accepted grill-session
  decisions and supporting research notes.
