# Agent Workspace Guide

This directory is the AI-readable planning and progress layer for C4OS.

Use:

- `.agents/specs/` for durable intent, requirements, decisions, risks,
  acceptance criteria, evidence, and source-retirement decisions.
- `.agents/progress/` for active execution state, item packets, logs, outputs,
  and handoffs.

Do not add new updates to `.task-bank/`. It is deprecated after migration to
this directory.
